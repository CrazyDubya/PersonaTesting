# Persona Testing

This repository contains the `persona_eval` project that reproduces and expands a persona-based evaluation harness for GPQA and MMLU-Pro.

Key entrypoints:
- `persona_eval/README.md` for detailed instructions.
- `persona_eval/config/default_config.yaml` for experiment settings.
- `persona_eval/src/cli.py` to run the full experiment with `python -m src.cli` from inside the `persona_eval` folder.
